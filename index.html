What SX2 does

Everything your server needs

Three powerful systems working together to keep your community running smoothly.

🎫

Ticket System

One-click private ticket channels. Users get instant help, staff get notified — no DM chaos.

💡

Suggestion Voting

Let your community speak up. Suggestions get posted with 👍 👎 reactions automatically.

🛠️

Smart Command Help

When a command fails, SX2 explains why and shows the correct usage — not a cold error.

⚡

Instant Response

Tickets open in milliseconds. Background tasks handle staff DMs so nothing blocks the user.

🔒

Private Channels

Each ticket is visible only to the user and your staff team — full privacy by design.

🧩

Easy Setup

Simple admin commands to configure channels and roles. Up and running in under a minute.

Reference

All commands

Every command SX2 supports, at a glance.

!suggest <message> Submit a suggestion to the community voting channel User
!help Get a list of all available commands User
!help <command> Get detailed help for a specific command User
Create Ticket Button Click the button in your support channel to open a private ticket instantly User
Close Ticket Button Click inside your ticket channel to close it when your issue is resolved User
!setup tickets <#channel> Set the channel where the ticket creation button will appear Admin
!setup suggestions <#channel> Set the channel where community suggestions are posted Admin
!setup staff <@role> Set the staff role that gets notified and can access all tickets Admin
!close Force close a ticket from inside the ticket channel Admin
!ticket panel Resend the ticket creation panel button to the current channel Admin
